Q: What does `O(n)` mean?
A: Linear time complexity — the runtime grows proportionally to the input size.

---

Q: What is a closure?
A: A function that captures variables from its enclosing scope.

---

Q: What is the difference between a stack and a queue?
A: A stack is LIFO (last in, first out). A queue is FIFO (first in, first out).

Q: What is a hash table's average-case time complexity for lookups?
A: $O(1)$ — constant time.

---

Q: What is the difference between `==` and `===` in JavaScript?
A: `==` performs type coercion before comparison. `===` checks both value and type without coercion.
